# Code of Conduct

## Our Commitment

This project is committed to maintaining a **respectful, serious, and constructive
environment** for collaboration.

The repository exists to document and organize **historical and canonical information**
about Marian apparitions based on official sources.

Disagreement is natural. Disrespect is not.

---

## Expected Behavior

All participants are expected to:

- Communicate respectfully and professionally
- Focus discussions on facts, sources, and methodology
- Accept constructive feedback
- Respect different cultural, academic, and religious backgrounds
- Assume good faith in the absence of clear evidence otherwise

---

## Unacceptable Behavior

The following behaviors are not tolerated:

- Harassment, insults, or personal attacks
- Aggressive, mocking, or dismissive language
- Ideological hostility or provocation
- Disruptive behavior that derails technical or editorial discussions
- Repeated attempts to impose personal beliefs as authoritative sources

---

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub issues
- Pull requests
- Commit messages
- Discussions
- External communications directly related to the project

---

## Editorial Neutrality

This project documents **historical claims and ecclesiastical classifications**.

It does **not**:
- Endorse personal belief systems
- Promote devotional practices
- Resolve theological disputes

Contributors must distinguish clearly between:
- Documented facts
- Official ecclesiastical positions
- Personal opinions

Personal opinions must not be presented as project content.

---

## Enforcement

Maintainers are responsible for enforcing this Code of Conduct.

Violations may result in:

- Requests for correction
- Removal of comments or contributions
- Temporary or permanent bans from participation

Enforcement decisions are made to protect the integrity of the project and its community.

---

## Reporting Issues

If you experience or witness behavior that violates this Code of Conduct, please report it
privately via the contact information provided in `SECURITY.md`.

Reports will be handled discreetly and respectfully.
